thinkPHP中M方法和D方法的区别:

M方法实例化模型无需用户为每个数据表定义模型类，如果D方法没有找到定义的模型类，则会自动调用M方法。通俗一点说：M实例化参数是数据库的表名。D实例化的是你自己在Model文件夹下面建立的模型文件。

Git使用中的撤销操作

发表于 2017-06-01 | 分类于 git

在git的使用过程中，有的时候操作会有失误，会进行撤销操作，下面是两个总结：

  1. 为文件进行了操作，但是没有什么实质性的修改，使用 git status时，会发现文件一致是 modified 状态，这是我们想要撤销，恢复到修改前的版本
    • 这一步一定要确定这个文件缺失是要恢复到修改前的版本！
      git checkout -- 文件名
  2. 撤销 git add 添加到暂存区的 操作：
    当使用git add * 将全部修改的文件加入了暂存区域， 要撤销自己不想加入的文件时，使用以下命令
    git reset HEAD 文件名可以将 modified 文件撤回
