配置Nginx反向代理

安装brew

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装Nginx

brew install nginx

启动Nginx

brew services start nginx

进入浏览器，打开localhost:8000 有欢迎页面表示安装成功

配置Nginx

1: 增加一个server

2: 重启Nginx

sudo nginx -s reload

退出Nginx

快速停止命令 sudo nginx -s stop

平稳退出命令：sudo nginx -s quit
